# Guess-My-Number (JavaScript)
<p>This repository contains a simple "Guess the Number" game implemented in JavaScript. The game prompts the player to guess a randomly generated number within the range of 1 to 20. It provides feedback to the player on whether their guess is too high, too low, or correct. The player can continue guessing until they guess the correct number or choose to reset the game.</p>

# Features
<li>Randomly generated number within the range of 1 to 20.</li>
<li>User input validation to ensure entered values are within the acceptable range.</li>
<li>Feedback messages for the player indicating if their guess is too high, too low, or correct.</li>
<li>Option to reset the game and play again.</li>
<li>Simple and intuitive user interface.</li>

# Purpose
<p>As I embark on my journey to master JavaScript, I'm focusing on honing my skills in DOM manipulation. This project serves as a playground for me to experiment with various DOM manipulation techniques, such as selecting elements, updating content, handling events, and creating interactive user interfaces. </p>
